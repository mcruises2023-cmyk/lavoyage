# Источники и проверка фактов для лендинга Explora Journeys

Дата проверки: 23.06.2026.

## Использовано для текстов и структуры
- официальный сайт Explora Journeys: позиционирование **Ocean State of Mind**, формат **Ultra-elegant Home at Sea**, inclusions и акция **Invitation to Discover**;
- подборка круизов Explora Journeys на **lavoyage.ru**: маршруты, даты, цены, ссылки на карточки круизов;
- предоставленная ссылка на акцию LaVoyage: `https://lavoyage.ru/sea_cruise/actions/action_details/?action_id=8246` — используется как основной CTA на акцию;
- референсный сайт PAC: `https://www.pac.ru/promo/explorajourneys/` — использован как визуальный ориентир по premium-стилистике.

## Использовано для изображений
- изображения `story-suite.jpg`, `story-dining.jpg`, `story-ocean.jpg`, `journey-sea.jpg`, `journey-ship.jpg` подготовлены на основе визуальных материалов официального сайта Explora Journeys;
- изображения `manager-1.jpg`, `manager-2.jpg`, `manager-3.jpg` сгенерированы специально для финального блока как иллюстративные лица менеджеров;
- фоновое видео первого экрана оптимизировано из предоставленного файла `EJ_HP_Video_D_2.mp4`: создана веб-версия `assets/explora-hero.mp4` и постер `assets/explora-poster.jpg`.

## Важно
- срок и условия акции могут меняться;
- цены в карточках указаны как ориентир «от» и требуют проверки наличия мест и актуальной стоимости на момент бронирования;
- финальные условия бронирования, применимость акции и наличие конкретных категорий сьютов должны подтверждаться менеджером ЛаВояж.

## Адаптивность
- В версии от 23.06.2026 добавлены отдельные CSS-правила для экранов 640 px, 380 px и 340 px: уменьшен мобильный H1, убраны потенциальные источники горизонтального overflow, сброшены отступы `figure`, включены переносы текста и адаптивное поведение кнопок/карточек.
